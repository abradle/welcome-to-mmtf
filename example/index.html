--- layout: page --- Service API provides programmatic access to PDB structures in MMTF format. You can interact with the service by making HTTP requests to our service. With every request you must send an URL with the following parameters:
http://mmtf.rcsb.org/full/4cup
This will return an encoded PDB structure as a msgpack bin. This example shows how to decode a PDB structure and get a number of chains from the decoded mmtf object:

  • Java
    var raw = decodeMsgpack( new Uint8Array( bin ) );
    var mmtfObject = decodeMmtf( raw );
    var structure = new SimpleStructure( mmtfObject );
    var numChains = structure.numChains;
